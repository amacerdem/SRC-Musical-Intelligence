### Dimension Names (Always Use These)

**6D Psychology (Turkish / English):**
- Enerji / Energy — index 0: loud/intense ↔ quiet/gentle
- Duygu Tonu / Valence — index 1: happy ↔ sad
- Hız / Tempo — index 2: fast ↔ slow
- Gerilim / Tension — index 3: strain ↔ release
- Hareket / Groove — index 4: want to move ↔ still
- Yoğunluk / Density — index 5: a lot going on ↔ sparse

**12D Cognition (Turkish / English):**
- Melodi / Melody, Armoni / Harmony, Ritim / Rhythm, Tını / Timbre
- Duygu / Emotion, Sürpriz / Surprise, İvme / Momentum, Hikaye / Story
- Tanıdıklık / Familiarity, Haz / Pleasure, Mekan / Space, Tekrar / Repetition

**Polarity Descriptors:**
| 6D | High | Low | Rising | Falling |
|----|------|-----|--------|---------|
| Energy | fully charged, intense | gentle and calm | building up | releasing |
| Valence | bright, joyful | dark, melancholic | lightening up | darkening |
| Tempo | fast, driving | slow, floating | accelerating | decelerating |
| Tension | strained, unresolved | resolved, relaxed | tightening | releasing |
| Groove | body wants to move | still, floating | locking in | loosening |
| Density | a lot going on | sparse, minimal | filling up | thinning out |

### Neurochemical Language

Always accessible, any tier:
- **Dopamine (DA)** — "the thrill of surprise" — anticipation, wanting, prediction error
- **Norepinephrine (NE)** — "your curiosity engine" — arousal, alertness, attention
- **Endogenous Opioids (OPI)** — "the warmth of beauty" — pleasure, liking, chills
- **Serotonin (5-HT)** — "patience and comfort" — familiarity, contentment, mood regulation
