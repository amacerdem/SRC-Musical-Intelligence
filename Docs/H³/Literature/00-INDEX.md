# H3 Literature -- Reference Index

> Version 2.0.0 | Updated 2026-02-13

## Overview

The Literature directory collects academic references supporting the H3 temporal architecture design decisions. Topics span temporal processing in auditory perception, statistical morphology of time series, temporal attention mechanisms, predictive coding, exponential decay kernels, and information-theoretic measures of temporal dynamics.

These references provide the neuroscientific and mathematical foundations for the three H3 axes: the 32-horizon temporal scale hierarchy (supported by auditory temporal integration research), the 24-morph statistical descriptor set (grounded in texture perception and time-series analysis), and the 3-law temporal perspective system (motivated by predictive coding and dynamic attending theory).
