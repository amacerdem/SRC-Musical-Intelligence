## Core Thesis

> The MI system speaks ONE language: **spectral**.
> R³ decomposes. H³ integrates. C³ interprets.
> No layer uses traditional music theory as a computational primitive.
> "Chord", "key", "cadence" are C³-emergent concepts, not R³/H³ inputs.

## Design Principles

1. **R³ = Microscope** — 5.8ms spectral snapshots, purely physical/psychoacoustic
2. **H³ = Temporal Lens** — Horizons give R³ data temporal context → first meaning layer
3. **R³+H³ = Temporal Perceptual Events** — Named, traceable, spectral
4. **C³ = Brain** — Consumes R³+H³ events, derives higher-order concepts internally
5. **Provenance is mandatory** — Every value C³ sees traces back to R³ source + H³ processing
