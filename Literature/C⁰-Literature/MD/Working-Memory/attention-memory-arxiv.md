## Abstract

This review synthesizes evidence on how neural oscillations, particularly in the alpha (8-12 Hz) and theta (4-8 Hz) frequency bands, reflect cognitive and memory performance. Good performance is related to two types of EEG phenomena: (i) a tonic increase in alpha but a decrease in theta power, and (ii) a large phasic (event-related) decrease in alpha but increase in theta, depending on the type of memory demands. The encoding of new information is reflected by theta oscillations in hippocampo-cortical feedback loops, whereas search and retrieval processes in long-term memory are reflected by upper alpha oscillations in thalamo-cortical feedback loops.

## Key Findings

### Alpha Oscillations (8-12 Hz)

1. **Functional Inhibition Hypothesis**
   - Alpha power increase = functional inhibition
   - Suppresses task-irrelevant processing
   - Allocates resources to relevant regions
   - Active gating mechanism

2. **Upper vs. Lower Alpha**
   - Lower alpha (8-10 Hz): Attentional processes
   - Upper alpha (10-12 Hz): Semantic processing
   - Different functional roles
   - Different topographic patterns

3. **Memory Functions**
   - Alpha desynchronization during encoding
   - Alpha synchronization during inhibition
   - Retrieval involves alpha decrease
   - Individual alpha frequency predicts performance

### Theta Oscillations (4-8 Hz)

1. **Memory Encoding**
   - Frontal midline theta during working memory
   - Hippocampal theta for episodic memory
   - Theta phase resets to stimuli
   - Theta power increases with memory load

2. **Working Memory Maintenance**
   - Sustained theta during delay period
   - Frontal-posterior theta coherence
   - Load-dependent theta increase
   - Executive control association

3. **Cross-Frequency Coupling**
   - Theta-gamma coupling for WM
   - Nested oscillation architecture
   - Sequential item representation
   - Phase coding of multiple items

### Attention-Memory Interaction

1. **Attention Modulates Memory**
   - Selective attention enhances encoding
   - Alpha lateralization for spatial attention
   - Theta for sustained attention
   - Working memory capacity and attention

2. **Neural Mechanisms**
   - Thalamo-cortical alpha loops
   - Hippocampo-cortical theta loops
   - Prefrontal control of oscillations
   - Bottom-up and top-down interactions

## Key Equations

### Alpha Power Calculation
```
P_alpha = ∫_{8}^{12} |X(f)|² df
```
Where:
- X(f) = Fourier transform of EEG signal
- Integration over alpha band (8-12 Hz)

### Event-Related Desynchronization (ERD)
```
ERD% = (R - A) / R × 100
```
Where:
- R = reference period power
- A = activation period power
- Negative values = synchronization (ERS)

### Phase-Amplitude Coupling
```
MI = (H_max - H) / H_max
```
Where:
- H = entropy of amplitude distribution across phases
- H_max = maximum possible entropy
- MI = modulation index
