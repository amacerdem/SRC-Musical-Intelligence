## 2. Belief Categories

### Core (C) — Full Bayesian PE Cycle

Core beliefs undergo predict → observe → update every frame:

```
predicted = τ × prev + (1−τ) × baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = π_obs / (π_obs + π_pred + ε)
posterior = (1 − gain) × predicted + gain × observed
PE        = observed − predicted  →  feeds reward formula
```

F4 Core beliefs have the **HIGHEST τ values in the system** (0.7–0.85), reflecting memory's slow, persistent nature. Unlike F3 attention beliefs (τ=0.25–0.4) that respond rapidly, memory beliefs change gradually — a retrieved memory does not vanish between frames.

MEAMN Core beliefs:
- **`autobiographical_retrieval`** (τ=0.85 — **highest τ in the system**, memory retrieval is maximally persistent)
- **`nostalgia_intensity`** (τ=0.8 — very high inertia, nostalgia builds slowly and lingers)
- **`emotional_coloring`** (τ=0.75 — high inertia, affective tags on memories are stable)

HCMC Core belief:
- **`episodic_encoding`** (τ=0.7 — high inertia, encoding state reflects sustained hippocampal binding)

### Appraisal (A) — Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no τ. They are evaluative judgments tracked in BeliefStore.

MEAMN Appraisals: **`retrieval_probability`**, **`memory_vividness`**, **`self_relevance`**
MMP Appraisals: **`melodic_recognition`**, **`memory_preservation`**
HCMC Appraisals: **`episodic_boundary`**, **`consolidation_strength`**

### Anticipation (N) — Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

MEAMN Anticipation: **`vividness_trajectory`**
MMP Anticipation: **`memory_scaffold_pred`**

## 4. Phase Schedule

F4 models operate in **Phase 2** (after F1's Phase 0 and F2/F3's Phase 1):

```
Phase 0a: BCH ∥ HMCE ∥ SNEM ∥ MEAMN ∥ MPG ∥ SRP ∥ PEOM  (7 independent relays)
Phase 0b: DAED (cross: BCH+MEAMN), HMCE (cross: reads other relays)
Phase 0c: Consonance multi-scale (BCH→HTP), Tempo multi-scale (PEOM→HMCE)
Phase 1:  F2 prediction + F3 attention (parallel)
Phase 2:  F4 memory + F5 emotion (parallel)
  ├── F4: MEAMN relay outputs available from Phase 0a
  │        └── HCMC, MMP mechanism models run here
  │              └── Remaining β/γ models
  └── F4 Core beliefs: predict/observe/update cycle
Phase 2a: Predict/Observe for all Core beliefs (including F4's 4 Core)
Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  Reward
```

**MEAMN relay** runs in Phase 0a (earliest). F4 mechanism models run in Phase 2.
F4 Core beliefs' full Bayesian cycle completes at Phase 2c.
