## 2. Belief Categories

### Core (C) — Full Bayesian PE Cycle

Core beliefs undergo predict → observe → update every frame:

```
predicted = τ × prev + (1−τ) × baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = π_obs / (π_obs + π_pred + ε)
posterior = (1 − gain) × predicted + gain × observed
PE        = observed − predicted  →  feeds reward formula
```

HTP Core beliefs:
- **`prediction_hierarchy`** (τ=0.4 — moderate inertia, tracks hierarchical structure shifts)
- **`prediction_accuracy`** (τ=0.5 — high inertia, slow-changing confidence in prediction quality)

SPH Core belief:
- **`sequence_match`** (τ=0.45 — moderate-high inertia, tracks memory template matching)

ICEM Core belief:
- **`information_content`** (τ=0.35 — lower inertia, responds quickly to surprise events)

### Appraisal (A) — Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no τ. They are evaluative judgments tracked in BeliefStore.

HTP Appraisal: **`hierarchy_coherence`**
SPH Appraisals: **`error_propagation`**, **`oscillatory_signature`**
ICEM Appraisals: **`defense_cascade`**, **`arousal_scaling`**, **`valence_inversion`**

### Anticipation (N) — Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

HTP Anticipations: **`abstract_future`**, **`midlevel_future`**
SPH Anticipation: **`sequence_completion`**
ICEM Anticipations: **`arousal_change_pred`**, **`valence_shift_pred`**

### Key Principle: Mechanism ≠ Belief

- **Mechanism** (mechanism/ folder): HOW the computation works (R³ inputs, H³ integration, weighted sums)
- **Belief** (this folder): WHAT the listener cognitively infers ("I expect this note", "this event is surprising")

## 4. Phase Schedule

F2 models operate in **Phase 1** (after F1's Phase 0):

```
Phase 0a: BCH ∥ HMCE ∥ SNEM ∥ MEAMN ∥ MPG ∥ SRP ∥ PEOM  (F1/other relays)
Phase 0b: DAED (cross: BCH+MEAMN), HMCE (cross)
Phase 0c: Consonance multi-scale (BCH→HTP), Tempo multi-scale (PEOM→HMCE)
Phase 1:  F2 prediction engine
  ├── HTP ∥ SPH ∥ ICEM  (3 α relays, parallel)
  ├── PWUP → WMED → UDP  (β chain)
  ├── CHPI (reads all α + PWUP + WMED)
  └── IGFE → MAA → PSH  (γ chain)
Phase 2a: Predict/Observe for all Core beliefs (including F2's 4 Core)
Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  Reward
```

F2 beliefs are available from Phase 1 onward. Core beliefs' full Bayesian cycle completes at Phase 2c.
