## 2. Belief Categories

### Core (C) -- Full Bayesian PE Cycle

Core beliefs undergo predict -> observe -> update every frame:

```
predicted = tau * prev + (1-tau) * baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = pi_obs / (pi_obs + pi_pred + eps)
posterior = (1 - gain) * predicted + gain * observed
PE        = observed - predicted  ->  feeds reward formula
```

F8 has the **HIGHEST tau values in all of C3** (0.88--0.95), reflecting the extremely slow accumulation of expertise and long-term plasticity:

TSCP Core belief:
- **`trained_timbre_recognition`** (tau=0.90 -- very high inertia, timbre recognition is deeply learned and stable; years of instrument training)

ESME Core belief:
- **`expertise_enhancement`** (tau=0.92 -- very high inertia, MMN amplification reflects years of training sharpening pre-attentive processing)

EDNR Core belief:
- **`network_specialization`** (tau=0.95 -- **highest tau in all C3**, network compartmentalization changes over months/years, not frames)

SLEE Core belief:
- **`statistical_model`** (tau=0.88 -- high inertia, statistical learning models build over extended exposure; fastest of F8 Core beliefs)

### Appraisal (A) -- Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no tau. They are evaluative judgments tracked in BeliefStore.

TSCP Appraisals: **`plasticity_magnitude`**
ESME Appraisals: **`pitch_mmn`**, **`rhythm_mmn`**, **`timbre_mmn`**
EDNR Appraisals: **`within_connectivity`**
SLEE Appraisals: **`detection_accuracy`**, **`multisensory_binding`**
ECT Appraisals: **`compartmentalization_cost`**

### Anticipation (N) -- Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

ESME Anticipation: **`expertise_trajectory`**
ECT Anticipation: **`transfer_limitation`**

          F8 reads: F2 prediction_accuracy, F4 consolidation_strength

**No F8 relay exists** in the current kernel -- F8 is evidence-only (deferred Phase 5).

## 6. Observe Formulas

### F8: trained_timbre_recognition (TSCP)

```
# TSCP mechanism outputs:
value = 0.40 * f01_trained_timbre_response + 0.30 * timbre_identity + 0.30 * recognition_quality

# f01 = sigma(0.35 * trist_balance + 0.35 * (1-inharm) * tonalness)
# timbre_identity = sigma(0.40 * trist_balance + 0.30 * (1-inharm) + 0.30 * spectral_autocorr)
# recognition_quality = spectral_envelope x instrument_identity

# Precision: 1/(std(f01, timbre_identity, recognition_quality) + 0.1)
```

### F8: expertise_enhancement (ESME)

```
# ESME mechanism outputs:
value = 0.50 * f04_expertise_enhancement + 0.30 * mmn_expertise_function + 0.20 * developmental_trajectory

# f04 = sigma(alpha * max(f01_pitch, f02_rhythm, f03_timbre))
# mmn_expertise_fn = sqrt(f04 * max(f01, f02, f03))
# developmental_trajectory = sigma(0.6 * f04 + 0.4 * mmn_expertise_fn)

# Precision: mmn_expertise_fn * expertise_alpha / (H3_std + eps)
```

### F8: network_specialization (EDNR)

```
# EDNR mechanism outputs:
value = 0.40 * f03_compartmentalization + 0.30 * f04_expertise_signature + 0.30 * network_architecture

# f03 = f01_within / (f02_between + eps)  -- compartmentalization ratio
# f04 = sigma(0.35 * tonalness_mean + 0.35 * pleasantness_mean)
# network_architecture = sigma(0.50 * f01 + 0.50 * f02)

# Precision: f04 * network_architecture / (H3_std + eps)
```

### F8: statistical_model (SLEE)

```
# SLEE mechanism outputs:
value = 0.40 * f01_statistical_model + 0.30 * exposure_model + 0.30 * pattern_memory

# f01 = sigma(0.35 * loudness_mean + 0.35 * amplitude_entropy)
# exposure_model = EMA of f01 over session timescale
# pattern_memory = EMA of pitch_stability with tau=3s

# Precision: 1/(std(f01, exposure_model, pattern_memory) + 0.1)
```
