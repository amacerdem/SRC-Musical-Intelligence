## 2. Belief Categories

### Core (C) — Full Bayesian PE Cycle

Core beliefs undergo predict → observe → update every frame:

```
predicted = τ × prev + (1−τ) × baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = π_obs / (π_obs + π_pred + ε)
posterior = (1 − gain) × predicted + gain × observed
PE        = observed − predicted  →  feeds reward formula
```

F5 Core beliefs have **moderate τ values** (0.5–0.65), reflecting the timescale of emotional perception. Unlike F4 memory beliefs (τ=0.7–0.85) that persist for seconds, emotion beliefs update at phrase-level — slow enough to require harmonic context but fast enough to track modulations and dynamic changes.

VMM Core beliefs:
- **`perceived_happy`** (τ=0.55 — moderate inertia, tracks major/consonant character over phrases)
- **`perceived_sad`** (τ=0.55 — symmetric with perceived_happy, tracks minor/dissonant character)

AAC Core belief:
- **`emotional_arousal`** (τ=0.5 — lowest F5 inertia, arousal responds faster than valence perception)

NEMAC Core belief:
- **`nostalgia_affect`** (τ=0.65 — highest F5 inertia, nostalgia builds slowly and lingers like memory)

### Appraisal (A) — Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no τ. They are evaluative judgments tracked in BeliefStore.

VMM Appraisals: **`mode_detection`**, **`emotion_certainty`**, **`happy_pathway`**, **`sad_pathway`**
AAC Appraisals: **`chills_intensity`**, **`ans_dominance`**
NEMAC Appraisals: **`self_referential_nostalgia`**, **`wellbeing_enhancement`**

### Anticipation (N) — Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

AAC Anticipation: **`driving_signal`**
NEMAC Anticipation: **`nostalgia_peak_pred`**

### Key Principle: Mechanism ≠ Belief

- **Mechanism** (mechanism/ folder): HOW the computation works (R3 inputs, H3 integration, weighted sums)
- **Belief** (this folder): WHAT the listener cognitively infers ("this sounds happy", "I am aroused")

## 4. Phase Schedule

F5 models operate in **Phase 2** (parallel with F4):

```
Phase 0a: BCH ∥ HMCE ∥ SNEM ∥ MEAMN ∥ MPG ∥ SRP ∥ PEOM  (7 independent relays)
Phase 0b: DAED (cross: BCH+MEAMN), HMCE (cross: reads other relays)
Phase 0c: Consonance multi-scale (BCH→HTP), Tempo multi-scale (PEOM→HMCE)
Phase 1:  F2 prediction + F3 attention (parallel)
Phase 2:  F4 memory + F5 emotion (parallel)
  ├── F5: VMM, AAC, NEMAC mechanism models run here
  │        └── VMM reads: MEAMN emotional_color + consonance PE + R³ mode cues
  │        └── AAC reads: SRP shared arousal + salience signal
  │        └── NEMAC reads: MEAMN memory state + familiarity
  └── F5 Core beliefs: predict/observe/update cycle
Phase 2a: Predict/Observe for all Core beliefs (including F5's 4 Core)
Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  Reward
```

**F5 mechanism models** run in Phase 2 after relays are available from Phase 0a.
F5 Core beliefs' full Bayesian cycle completes at Phase 2c.
