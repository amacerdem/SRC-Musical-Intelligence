## 2. Belief Categories

### Core (C) — Full Bayesian PE Cycle

Core beliefs undergo predict → observe → update every frame:

```
predicted = τ × prev + (1−τ) × baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = π_obs / (π_obs + π_pred + ε)
posterior = (1 − gain) × predicted + gain × observed
PE        = observed − predicted  →  feeds reward formula
```

BCH's Core belief: **`harmonic_stability`** (τ = 0.3 — low inertia, responds quickly to harmonic changes)

### Appraisal (A) — Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no τ. They are evaluative judgments tracked in BeliefStore.

BCH's Appraisal beliefs: **`interval_quality`**, **`harmonic_template_match`**

### Anticipation (N) — Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

BCH's Anticipation belief: **`consonance_trajectory`**

### Key Principle: Mechanism ≠ Belief

- **Mechanism** (mechanism/ folder): HOW the computation works (R³ inputs, H³ integration, weighted sums)
- **Belief** (this folder): WHAT the listener cognitively infers ("this sound is stable", "this interval is consonant")

BCH computes 16D. Only 4 of those dimensions produce beliefs. The rest (E0, E1, E3, M0–M3, P2, P3, F1–F3) are either computational intermediates or cross-model outputs feeding PSCL, PCCR, SRP, MEAMN.

## 4. Phase Schedule

BCH operates in **Phase 0a** (independent relay):

```
Phase 0a: BCH ∥ HMCE ∥ SNEM ∥ MEAMN ∥ MPG ∥ SRP ∥ PEOM  (7 independent relays)
Phase 0b: DAED (cross: BCH+MEAMN), HMCE (cross: reads other relays)
Phase 0c: Consonance multi-scale (BCH→HTP), Tempo multi-scale (PEOM→HMCE)
Phase 1:  Salience (uses SNEM+SRP+MPG)
Phase 2a: Predict/Observe for all Core beliefs
Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  Reward
```

BCH beliefs are available from Phase 0a onward. `harmonic_stability`'s full Bayesian cycle completes at Phase 2c.
