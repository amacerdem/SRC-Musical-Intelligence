## 2. Belief Categories

### Core (C) вҖ” Full Bayesian PE Cycle

Core beliefs undergo predict вҶ’ observe вҶ’ update every frame:

```
predicted = П„ Г— prev + (1вҲ’П„) Г— baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = ПҖ_obs / (ПҖ_obs + ПҖ_pred + Оө)
posterior = (1 вҲ’ gain) Г— predicted + gain Г— observed
PE        = observed вҲ’ predicted  вҶ’  feeds reward formula
```

SNEM Core beliefs:
- **`beat_entrainment`** (П„=0.35 вҖ” moderate inertia, tracks beat-locking at ~300ms)
- **`meter_hierarchy`** (П„=0.4 вҖ” slightly higher inertia, meter structure changes slower than beat)

IACM Core belief:
- **`attention_capture`** (П„=0.25 вҖ” **lowest П„ in F3**, responds fastest to inharmonic events)

CSG Core belief (cross-function from F1):
- **`salience_network_activation`** (П„=0.3 вҖ” low inertia, rapid salience response to consonance changes)

### Appraisal (A) вҖ” Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no П„. They are evaluative judgments tracked in BeliefStore.

SNEM Appraisal: **`selective_gain`**
IACM Appraisals: **`object_segregation`**, **`precision_weighting`**
CSG Appraisals: **`sensory_load`**, **`consonance_valence_mapping`**
AACM Appraisals: **`aesthetic_engagement`**, **`savoring_effect`**

### Anticipation (N) вҖ” Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

SNEM Anticipations: **`beat_onset_pred`**, **`meter_position_pred`**
IACM Anticipation: **`attention_shift_pred`**
CSG Anticipation: **`processing_load_pred`**

## 4. Phase Schedule

F3 models operate in **Phase 1** (same as F2, after F1's Phase 0):

```
Phase 0a: BCH вҲҘ HMCE вҲҘ SNEM вҲҘ MEAMN вҲҘ MPG вҲҘ SRP вҲҘ PEOM  (7 independent relays)
Phase 0b: DAED (cross: BCH+MEAMN), HMCE (cross: reads other relays)
Phase 0c: Consonance multi-scale (BCHвҶ’HTP), Tempo multi-scale (PEOMвҶ’HMCE)
Phase 1:  F2 prediction + F3 attention (parallel)
  в”ңв”Җв”Җ F2: HTP вҲҘ SPH вҲҘ ICEM  (3 Оұ relays)
  в”Ӯ        в””в”Җв”Җ PWUP вҶ’ WMED вҶ’ UDP вҶ’ CHPI  (ОІ chain)
  в”Ӯ              в””в”Җв”Җ IGFE* вҶ’ MAA вҶ’ PSH  (Оі chain, *IGFE is F3)
  в”ңв”Җв”Җ F3: SNEM вҲҘ IACM  (2 Оұ models, parallel with F2)
  в”Ӯ        в””в”Җв”Җ BARM, STANM, AACM, AMSS, ETAM  (ОІ depth 1)
  в”Ӯ              в””в”Җв”Җ DGTP, SDL, NEWMD  (Оі depth 2)
  в””в”Җв”Җ Salience mixing (kernel): energy + HВі velocity + PE_carry + relay signals
Phase 2a: Predict/Observe for all Core beliefs (including F3's 4 Core)
Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  Reward
```

**SNEM relay** runs in Phase 0a (earliest). F3 mechanism models run in Phase 1.
F3 Core beliefs' full Bayesian cycle completes at Phase 2c.

**Note**: IGFE (PCU-Оі1) is F3-primary but runs within the F2 pipeline (PCU depth chain) due to its WMED dependency. It participates in F3 salience computation via its gamma binding outputs.
