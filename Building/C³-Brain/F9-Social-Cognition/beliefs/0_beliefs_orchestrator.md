## 2. Belief Categories

### Core (C) -- Full Bayesian PE Cycle

Core beliefs undergo predict -> observe -> update every frame:

```
predicted = tau * prev + (1-tau) * baseline + trend + periodicity + context
observed  = f(mechanism outputs)
gain      = pi_obs / (pi_obs + pi_pred + eps)
posterior = (1 - gain) * predicted + gain * observed
PE        = observed - predicted  ->  feeds reward formula
```

F9 has the **lowest tau values among Phase 3 functions** (0.60--0.65 vs F8's 0.88--0.95), reflecting social cognition's faster dynamics -- interpersonal synchrony fluctuates on the timescale of seconds, not the years-long accumulation of expertise:

NSCP Core belief:
- **`neural_synchrony`** (tau=0.65 -- moderate inertia, population-level neural synchronization adapts within seconds of coordinated listening onset)

DDSMI Core belief:
- **`social_coordination`** (tau=0.60 -- moderate inertia, partner tracking updates rapidly during dyadic interaction; visual contact modulation within ~100ms)

### Appraisal (A) -- Observe Only

Appraisal beliefs are computed directly by mechanisms. No predict/update cycle, no PE, no tau. They are evaluative judgments tracked in BeliefStore.

SSRI Appraisals: **`synchrony_reward`**, **`social_bonding`**, **`group_flow`**, **`entrainment_quality`**, **`social_prediction_error`**
DDSMI Appraisals: **`resource_allocation`**

### Anticipation (N) -- Forward Predictions

Anticipation beliefs are forward predictions generated by F-layer outputs. They feed Core beliefs' predict() method and the precision engine.

NSCP Anticipation: **`catchiness_pred`**
SSRI Anticipation: **`collective_pleasure_pred`**

```

Phase 2b: PE + Precision
Phase 2c: Update (Bayesian posterior)
Phase 3:  F8 Learning + F9 Social (parallel)
          F9 reads: F7 period_entrainment, F7 groove_quality
Phase 4:  PE + Precision for ALL Core Beliefs (including F9's 2 Core)
```

**No F9-dedicated relay exists** in the current kernel -- F9 is evidence-only (deferred Phase 5).

## 8. Social Reward Amplification

F9 introduces a key modulatory effect: social context amplifies individual reward by 1.3--1.8x baseline (Dunbar 2012, Tarr et al. 2014, Kokal et al. 2011). This is mediated by:

1. **Synchrony reward** (SSRI): Coordinated moments generate pleasure via endorphin/oxytocin release
2. **Social prediction error** (SSRI): Better-than-expected coordination produces reward surge
3. **Group flow** (SSRI): Shared absorption amplifies hedonic experience

The amplification ratio is computed as:
```
sync_amplification = 1.0 + f01_synchrony_reward * (f04_entrainment_quality + f02_social_bonding)
# Range: [1.0, ~3.0], typical: 1.3-1.8
```
