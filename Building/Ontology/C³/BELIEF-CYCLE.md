## 3 Belief Categories

C┬│ beliefs are **cognitive inferences** тАФ NOT signal features. R┬│/H┬│ produce signal features
(roughness, periodicity, spectral flux). Beliefs are what the brain INFERS from those signals.

Example: R┬│ `roughness_total = 0.3` is a signal feature. The belief `harmonic_stability = 0.7`
is the cognitive inference "this sound is harmonically resolved."

### Category Definitions

| Category | Symbol | Bayesian Cycle | PE | Description |
|----------|--------|---------------|-----|-------------|
| **Core** | C | Full (predictтЖТobserveтЖТupdate) | Yes | Primary cognitive states; ╧Д-controlled inertia |
| **Appraisal** | A | Observe-only (no predict) | No | Evaluative judgments computed by mechanisms |
| **Anticipation** | N | Prediction outputs | No | Forward predictions; feed Core predict() |

**Core Beliefs**: Full Bayesian cycle. Each has ╧Д (inertia), baseline, precision, and PE history.
These drive the reward formula via prediction error.

**Appraisal Beliefs**: Mechanism outputs tracked in BeliefStore. No predict/update cycle тАФ
the mechanism computes the value directly each frame. Serve as evidence and context for
Core Beliefs and downstream Functions.

**Anticipation Beliefs**: Forward predictions generated by mechanisms. Used as prediction
signals by Core Beliefs' predict() method. Also consumed by the precision engine for
╧А_pred estimation.

## Bayesian Belief Update (Core Beliefs Only)

Her Core belief her frame'de 3 ad─▒m (t├╝m Function'lar i├зin ayn─▒ d├╢ng├╝):

```
observed  = f(R┬│, H┬│, relay_outputs)     # measure
predicted = ╧Д ├Ч prev + (1тИТ╧Д) ├Ч baseline  # predict
             + w_trend ├Ч H┬│_M18(trend)
             + w_period ├Ч H┬│_M14(periodicity)
             + w_ctx ├Ч ╬г(context_weights ├Ч beliefs_{t-1})
posterior = (1 тИТ gain) ├Ч predicted + gain ├Ч observed   # update
```

**Bayesian gain:**
```
gain = ╧А_obs / (╧А_obs + ╧А_pred + ╬╡)
```
- `╧А_obs` = observation precision (sensory confidence, from observe())
- `╧А_pred` = prediction precision (from PE history)
- High `╧А_obs` тЖТ trust observation. High `╧А_pred` тЖТ trust prediction.

**Prediction Error (Core Beliefs only):**
```
PE = observed тИТ predicted
```

> Appraisal and Anticipation beliefs do NOT produce PE. They are direct mechanism outputs.

## Multi-Scale Prediction (v2.0)

Belief'ler birden fazla temporal horizon'da tahmin yapabilir.

**Scale-matched evidence weighting:**
```
w(h, ╬Ф) = exp(тИТ╬▒ |h тИТ ╬Ф|) / ╬г exp(тИТ╬▒ |h' тИТ ╬Ф|)
```
- `h` = evidence horizon, `╬Ф` = target horizon
- Peak weight when evidence matches target timescale
- ╬▒ = 0.3 (default decay): ┬▒3 index тЖТ %20, ┬▒8 тЖТ %5

**Per-horizon prediction:**
```
pred_h = ╧Д ├Ч prev + (1тИТ╧Д) ├Ч baseline + w_trend ├Ч ╬г(w(h_ev, h_target) ├Ч M18(h_ev))
```

**Aggregation (T_char-based):**
```
closest_h = argmin_h |HORIZON_SECONDS[h] тИТ T_char|
agg_weights = scale_matched_weights(closest_h, horizons, ╬▒)
posterior = ╬г agg_weights[h] ├Ч pred_h
```

> Multi-scale applies to Core Beliefs that have horizon bands.
> Appraisal/Anticipation beliefs operate at single-scale (their mechanism's native rate).
